XE typo fix addon
=================
[네이버 개발자센터](https://developers.naver.com/)에서 제공하는 
[네이버 오픈 API](https://developers.naver.com/docs/common/openapiguide/#/apilist.md/)를 이용하여
오픈소스 CMS [XpressEngine](https://xpressengine.com/)에서 검색어의 오타를 수정/추천해주는 애드온 입니다.
